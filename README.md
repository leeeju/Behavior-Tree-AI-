## Behavior_Tree 란?

비헤이비어 트리(BT)는 로봇이나 컴퓨터 게임의 가상 엔티티와 같은 자율 에이전트에서 서로 다른 작업 간의 전환을 구조화하는 방법이다, BT는 모듈식이면서 반응형인 복잡한 시스템을 만드는 매우 효율적인 
방법이며 이러한 속성은 컴퓨터 게임 프로그래밍에서 AI 및 로봇 공학의 많은 분야로 BT가 확산되며 다양한 응용 프로그램에서 매우 활용됩니다. 안전성, 견고성 및 효율성과 같은 속성은 자율 시스템에 중요
하며 BT의 상태 공간 설명을 사용하여 이러한 속성을 공식적으로 분석하기 위한 도구 세트를 설명합니다.
